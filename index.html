TypeScript Exercise - Tutti i Concetti Implementati

1 Tipi Primitivi

I tipi primitivi principali in TypeScript sono: string, number, boolean, null, undefined, symbol, bigint

2 Variabili Tipizzate

const mioNome: string = "Mario";
const miaEta: number = 25;
const stoStudiandoTypeScript: boolean = true;

3-6 Funzioni Tipizzate

const greet = (name: string): string => "Ciao " + name;
const sum = (a: number, b: number): number => a + b;
const calcolaPrezzoConIVA = (prezzo: number): number => prezzo * 1.22;
const concatenaStringhe = (str1: string, str2: string): number => (str1 + str2).length;

7-8 Type Union

type StringOrNumber = string | number;
let valore: number | null | undefined;

9 Stringhe Letterali

type GiorniSettimana = "Lunedì" | "Martedì" | "Mercoledì" | "Giovedì" | "Venerdì" | "Sabato" | "Domenica";

10-11 Array e Tuple

const numbers: number[] = [1, 2, 3];
type TuplaMista = [string, string, string, number, number];

12-14 Interfacce

interface Persona { firstname: string; lastname: string; age: number; }
interface Utente { email: string; telefono?: string; }

15-17 Oggetti e Estensioni

interface Veicolo { marca: string; modello: string; anno: number; }
interface Auto extends Veicolo { numeroPorte: number; tipoAlimentazione: string; }

18-20 Generics

interface ApiResponse {
  success: boolean;
  data: T;
  message?: string;
  timestamp: Date;
}
